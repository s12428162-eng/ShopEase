ShopEase هو متجر إلكتروني تفاعلي مصمم بعناية باستخدام React و TypeScript، يتيح للمستخدمين تصفح وشراء المنتجات بسهولة وسلاسة.

المميزات الرئيسية:

عرض المنتجات

صفحة عرض المنتجات تعرض جميع العناصر مع الاسم، الصورة، والسعر.

يمكن فرز المنتجات حسب الاسم (أبجدي) أو السعر.

صفحة تفاصيل المنتج

كل منتج يحتوي على صفحة تفاصيل تعرض الصورة، السعر، الوصف، والمقاسات المتاحة (S, M, L, XL).

يمكن للمستخدم اختيار المقاس والكمية قبل إضافة المنتج إلى السلة.

سلة التسوق (Cart)

تعرض المنتجات المضافة مع الاسم، المقاس، الكمية، والسعر.

يمكن زيادة أو تقليل الكمية أو إزالة المنتج من السلة مباشرة.

يظهر المجموع الكلي لجميع المنتجات المضافة.

البحث عن المنتجات

إمكانية البحث السريع عن المنتجات باستخدام الاسم.

عرض النتائج في واجهة سهلة القراءة.

التنقل السلس

شريط تنقل أعلى الصفحة يحتوي على روابط لجميع الأقسام (المنتجات، قمصان، جاكيتات، من نحن، المقالات).

أيقونة سلة التسوق تعرض عدد العناصر الحالية في السلة.

توافق مع الأجهزة المختلفة.
